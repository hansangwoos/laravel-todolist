## 프로젝트 개요

Laravel 프레임워크를 학습하며 개발한 완전한 CRUD 기능을 가진 TodoList 애플리케이션입니다
MVC패턴과 모던 웹 개발 기술을 적용하여 구현했습니다.

주요 기능
<table>
<tr>
<td width="50%">
🎯 핵심 기능

✅ 완전한 CRUD 시스템
✅ 할일 상태 관리 (완료/진행중)
✅ 실시간 통계 (전체/완료/진행중)
✅ 필터링 시스템 (상태별 분류)
✅ 검색 기능 (제목/내용 기반)

</td>
<td width="50%">
💎 고급 기능

🚀 실시간 미리보기 (작성/수정시)
⌨️ 키보드 단축키 지원
💾 자동 저장 기능 (LocalStorage)
📱 완전 반응형 디자인
🎨 모던한 UI/UX
